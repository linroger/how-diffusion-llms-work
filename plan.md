# Plan: Research Diffusion Models for Text and Coding

## Objective
Research diffusion models applied to text generation and code generation, with focus on Google DeepMind and Ant Group contributions. Collect papers, synthesize findings, and produce a comprehensive report.

## Stage 1 — Deep Research (deep-research-swarm)
- **Route**: Route A (Wide Search) — broad research area with multiple dimensions
- **Sub-tasks**:
  1. Research Google DeepMind's diffusion model work for text and code
  2. Research Ant Group's diffusion model work for text and code
  3. Research broader landscape of diffusion models for discrete sequences (text/code)
  4. Collect and download key research papers
- **Output**: Validated research brief with collected papers

## Stage 2 — Report Writing (report-writing)
- Synthesize findings into a structured report
- Include: overview, key contributions from each organization, technical approaches, comparisons, trends
- **Output**: Final markdown report

## Stage 3 — Artifact Production (docx)
- Convert final markdown report to Word document
- **Output**: .docx file delivered to user
